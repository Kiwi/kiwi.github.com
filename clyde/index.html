Clyde

Clyde is a replacement for Arch Linux's pacman. It is written in the Lua programming language and uses lualpm (a Lua binding to libalpm) as its backend. It is intended to be a unified replacement for other package management utilities (specifically it intends to replace powerpill and yaourt).

Features

  • Automatic installation of AUR packages
  • Downloading from multiple mirrors for increased speed
  • Colorized output

Curlpaste

Curlpaste is a utility to post text files to a number of pastebin sites using Curl and Lua.

Features

  • Support for several different pastebins.
    • Available services are codepad (default), ghost, ca, lodgeit, dpaste, and fpaste
  • Multiple file uploading in one command, by default a separate link for each but you may specify that Curlpaste should concat them into one paste.
  • Pasting from the X primary selection buffer as well as putting the returned url on the clipboard
  • Specify description, nick, expiration date, language, service, or let Curlpaste use the sensible default options.
  • Many more! See help.txt for a full list.
